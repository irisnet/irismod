# Parameters

The token module contains the following parameters:

| Key               | Type | Example                              |
| :---------------- | :--- | :----------------------------------- |
| TokenTaxRate      | Dec  | "0.4"                                |
| IssueTokenBaseFee | Coin | {"denom": "stake","amount": "60000"} |
| MintTokenFeeRatio | Dec  | "0.1"                                |
