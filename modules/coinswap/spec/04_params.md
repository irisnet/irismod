# Parameters

The coinswap module contains the following parameters:

| Key             | Type         | Example    |
| :-------------- | :----------- | :--------- |
| Fee             | string (dec) | "0.003"    |
| PoolCreationFee | sdk.Coin     | "5000iris" |
| TaxRate         | string (dec) | "0.4"      |
